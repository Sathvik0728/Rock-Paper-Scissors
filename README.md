# 🎮 Rock Paper Scissors – AI Battle Arena

A futuristic and responsive **Rock Paper Scissors** web game built using **Flask, HTML, CSS, and JavaScript** with a premium cyberpunk gaming interface.

---

## 🚀 Live Demo

🔗 https://rock-paper-scissors-b2t8.onrender.com

---

## ✨ Features

- 🎮 Interactive Rock Paper Scissors gameplay
- 🔥 Best Of 3 Mode
- ⚡ Best Of 5 Mode
- ♾️ Infinite Mode
- 🤖 AI Opponent
- 🎉 Winner Popup System
- 🚫 Mode Selection Validation Popup
- 🌌 Animated Cyberpunk Background
- 💎 Modern Glassmorphism UI
- 📱 Fully Responsive Design
- 💻 Desktop + Mobile Support

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sathvik0728/Rock-Paper-Scissors.git
```

### Open Project Folder

```bash
cd Rock-Paper-Scissors
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Flask Server

```bash
python app.py
```

### Open Browser

```bash
http://127.0.0.1:5000
```
